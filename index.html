Behind Nat

It's no fun being behind Nat

In the picture on the right, you're in Nat's shadow.

The Internet was built on the notion of every computer being able to communicate, no matter what network it was on. Connecting computers up that weren't connected was called Internetting.

If you want to show off a web app on your own computer, or run a file hosting service for yourself and your brother and sister, Nat can get in your way.

You can join the next-generation Internet, IPv6, even while behind Nat. There's no sign-up process and no permission is required. You just install some software, and every computer of yours can get a global IPv6 address. It's easy to switch it on.

Are you behind Nat?

You probably are. Try using Am I Behind Nat.com to see what it thinks.

If you never want to run network services on your computer, this website isn't much use to you. If you want to run network services that others can reach, even if temporarily, keep reading.
